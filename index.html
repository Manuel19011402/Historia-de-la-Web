1990 WEB 1.0

Fue la primera y en ella solo se podia consumir contenido. Se trataba de informacion a la que se podia acceder, pero sin la posibilidad de inteactuar: era unidireccional.



2000 - 2004 WEB 2.0

Permite la intereaccion entre usuarios. Da control total al usuario de manejar su informacion. Usuarios productores como consumidores Web colaboratuva. O'Really Media da por primera ves el termino Web 2.0 en octubre del 2004.



2010 WEB 3.0

Surge la Web 3.0 tambien conocida como la WEB semantica, esta renovo el sentido de la WEB proporcionandobusquedas mas inteligentes con ayuda de palabras clave, inteligencia artificial y reconocimiento de voz.



2016 WEB 4.0

La WEB 4.0 empezo en el 2016 y se centra en ofrecer un comportamiento mas inteligente y mas predictivo, de modo que podamo, con solo realizar una afirmacion o una llamada, poner en marcha un comjunto de acciones que tendran como resultado aquello que pedimos, deseamos o decimos.
